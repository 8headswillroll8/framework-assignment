# Orbit

A responsive social media application built with Tailwind CSS as part of the Noroff CSS Frameworks Course Assignment.

## Features

- Authentication page with HTML form validation
- Feed page with posts, thumbnails, search, sorting, and post creation form
- Profile page with user information, followers/following statistics, and post grid
- Responsive design for mobile and desktop
- Built using Tailwind CSS
- Vite development environment

## Prerequisites

- Node.js (v20+)
- npm

## Getting Started

### Installation

npm install

### Running the project

npm run dev

Open the project in your browser at:

http://localhost:5173

### Production Build

npm run build

## Available Scripts

- npm run dev - Start Vite development server
- npm run build - Create production build
